about=This plugin brings together three modules in a single panel:
    - AROME (forecasts): queries the AROME Overseas package API (Indian Ocean domain, 0.025° grid), with dynamic parameter discovery, regional or Réunion view, automatic styling, and animated time series via the Temporal Controller.
    - Radar (near real-time): queries the Radar package API, extracts the Réunion precipitation mosaic (Piton Villers + Colorado radars, 500 m resolution, 5-minute steps), manual refresh building a local sliding history.
    - Observations (weather stations): queries the Observations package API, displays hourly station measurements for Réunion Island, with native temporal animation (vector layer).
    A Météo-France API key (portail-api.meteofrance.fr, "API Key" mode) is required, shared across all three modules.

    Developed with the assistance of an AI tool (Claude, Anthropic), under human supervision and validation for each component.
