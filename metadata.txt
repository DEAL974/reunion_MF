about=Ce plugin regroupe trois modules dans un panneau unique :
    - AROME (prévisions) : interroge l'API paquets AROME Outre-Mer (Océan Indien, grille 0.025°), avec découverte dynamique des paramètres, vue régionale ou Réunion, styles automatiques, séries temporelles animées via le Temporal Controller.
    - Radar (temps réel) : interroge l'API Paquet Radar, extrait la mosaïque de précipitation Réunion (radars Piton Villers + Colorado, résolution 500 m, pas de 5 min), actualisation manuelle constituant un historique local glissant.
    - Observations (stations) : interroge l'API Paquet Observations, affiche les mesures horaires des stations météo de La Réunion sur 24h glissantes, avec animation temporelle native (vecteur).
    Une clé API Météo-France (portail-api.meteofrance.fr, mode "API Key") est nécessaire, partagée entre les trois modules.

    Développement réalisé avec l'assistance d'une IA (Claude, Anthropic), sous supervision et validation humaine de chaque composant.
